# asfathesaurus

Dataset with ASFA thesaurus descriptors.

## Installation
``` r
devtools::install_github("dswdejonge/asfathesaurus")
```

## Usage
``` r
# Load
library(asfathesaurus)

# View documentation
?asfathesaurus

# View first entries
head(asfathesaurus)

# Write to CSV
write.csv(asfathesaurus, file = thesaurus.csv)
```
